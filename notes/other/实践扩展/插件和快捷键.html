# vscode 快捷键

  • ctrl/command+shift+f 查找所有文件内容中的某个关键词

  • ctrl+p 快速查找文件

  • Option + Up 向上移动行

  • Option + Down 向下移动行

  • Option + Shift + Up 向上复制行

  • Option + Shift + Down 向下复制行

  • 重新打开关闭的编辑页面 ctrl/command+shift+T
