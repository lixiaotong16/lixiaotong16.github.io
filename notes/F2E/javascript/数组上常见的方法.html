    # 数组上常用的方法

    # 1. push/pop, unshift/shift

    这四种方法均对原数组进行了直接的修改

    • push: 向数组尾部添加元素，返回数组长度
    • pop: 删除数组中最后一个元素，返回被删除的元素
    • unshift: 向数组头部添加元素，返回数组的长度
    • shift: 删除数组头部的第一个元素，返回被删除的元素
    const a = [2, 5, 7, 9, 5, 9, 0, 2];
a.push(10);
a.pop();
a.unshift(10);
a.shift();

    # 2. splice/reverse/sort/slice/concat

    直接在原数组上进行操作的是：splice、reserve、sort 返回结果数组的是：slice、concat

    • splice：从数组添加/删除数据，并返回删除的数据数组

      //splice用法
const a = [2, 5, 7, 9, 5, 9, 0, 2];
//添加数据 将8,6,4从数组索引1处添加进去
a.splice(1, 0, 8, 6, 4); // []
a; // [2, 8, 6, 4, 5, 7, 9, 5, 9, 0, 2]

//删除数据 将索引2处数据删掉
a.splice(2, 1); // [6]
a; // [2, 8, 4, 5, 7, 9, 5, 9, 0, 2]

    # 6. Array.of/Array.from

    • Array.of: 将参数中所有值作为元素形成数组

      Array.of(1, "2", true); // [1, '2', true]
// 参数为空时返回空数组
Array.of(); // []
      1
      2
      3

    • Array.from：将类数组对象或可迭代对象转化为数组

      // 参数为数组,返回与原数组一样的数组
Array.from([1, 2]); // [1, 2]
// 参数含空位
Array.from([1, , 3]); // [1, undefined, 3]
// 对各参数进行处理
Array.from([1, 2, 3], (n) => n * 2); // [2, 4, 6]
// 通过类数组生成数组
Array.from({ length: 2, 0: "test", 1: "test2" }); // ['test', 'test2']
// 通过Map生成数组
const map = new Map();
map.set("key0", "value0");
map.set("key1", "value1");
Array.from(map); // [['key0', 'value0'],['key1','value1']]
// 通过Set生成数组
const set = new Set([1, 2, 3]);
Array.from(set); // [1, 2, 3]
// 字符串生成数组
const str = "abc";
console.log(Array.from(str)); // ["a", "b", "c"]
