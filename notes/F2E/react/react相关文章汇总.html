    总结： 当我们编写组件时，应该尽量将组件编写为纯函数。 对于组件中的副作用，首先应该明确： 是「用户行为触发的」还是「视图渲染后主动触发的」？ 对于前者，将逻辑放在 Event handlers 中处理。 对于后者，使用 useEffect 处理。 这也是为什么 useEffect 所在章节在新文档中叫做 Escape Hatches —— 大部分情况下，你不会用到 useEffect，这只是其他情况都不适应时的逃生舱。
